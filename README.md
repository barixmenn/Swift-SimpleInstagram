# Swift-Instagram

The Instagram clone I created using FireBase, TableView , ImagePicker and Tab bar allows you to upload and like photos.

## Installation

Clone the [project](https://github.com/barixmenn/Swift-Instagram)

## Tech Stack
    - Swift 5 
    - FireBase
